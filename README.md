# LE202
# สรุปความเข้าใจ
## Heading
หัวข้อหลักใช้ #
หัวข้อรองใช้ ##
หัวข้อย่อยใช้ ###
### Bold
**word** คือตัวหนา
### Italic
*word* คือตัวเอียง
### Strikethrough
~~word~~ คือขีด1ขีดทับตัวหนังสือ
### Bold and italic
***word*** คือตัวหนังสือหนาและเอียง
### Quoting Code
... คือจัดรูปของโค้ดให้เป็นบล็อกที่ต่างกัน
### Image
![word](link of image) คือไว้ใส่รูปภาพ
### Task list
[ ],[?] สร้างตัวอย่างให้เอาไว้เลือก
### Emoji
:word: ใส่อีโมจิต่างๆ
